      Security Model
      
            
      CryptoForge Pro processes all data exclusively in your browser using the Web Cryptography API. No data is ever transmitted to any server.
      
            
        
                
      • Native: SHA-256, SHA-512, HMAC, PBKDF2, AES-GCM, CSPRNG → Built-in WebCrypto
        • 
                
      • CDN Opt-In: SHA-3, MD5, RIPEMD-160, Bcrypt, Argon2 → External libraries loaded on demand
        • 
            
      
            
      External libraries use Subresource Integrity (SRI) verification. Toggle via the sidebar "Libs" switch.
